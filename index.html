Xin chào — Trang GitHub Pages của bạn

Đây là mẫu **index.html** đơn giản. Chỉnh sửa nội dung rồi đẩy (push) lên nhánh main ở thư mục gốc để GitHub Pages xuất bản.

Nội dung mẫu

Thêm nội dung HTML, CSS hoặc JavaScript vào file này. Ví dụ nút bên dưới giúp bật/tắt chế độ tối (chỉ minh họa):

Bật/tắt chế độ tối

Cách sử dụng

  1. Đặt file này tên index.html vào thư mục gốc của repository.
  2. Push lên nhánh main.
  3. Vào Settings → Pages → Deploy from a branch → chọn main và thư mục /(root) → Save.
  4. Chờ một vài phút, truy cập https://{username}.github.io/{repository}/.
