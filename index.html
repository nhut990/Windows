Giới thiệu HTML Cơ Bản

Trang này giúp bạn hiểu rõ từng ký hiệu trong mã HTML — ví dụ <, >, {}, () là gì và chúng được dùng ra sao.

1️⃣ Giải thích các ký hiệu thường gặp

  • < >: Dấu ngoặc nhọn dùng để bao thẻ HTML. Ví dụ <html> mở thẻ, </html> đóng thẻ.
  • { }: Dấu ngoặc nhọn dùng trong CSS hoặc JavaScript để bao nội dung quy định. Ví dụ: body { color: red; }
  • ( ): Dấu ngoặc tròn thường dùng trong hàm (function) trong JavaScript. Ví dụ: alert("Xin chào!")
  • " " hoặc ' ': Dấu ngoặc kép/đơn bao quanh chuỗi ký tự (text string).
  • ;: Dấu chấm phẩy kết thúc lệnh trong CSS hoặc JavaScript.
  • //: Ghi chú 1 dòng trong JavaScript (bị trình duyệt bỏ qua).
  • /* ... */: Ghi chú nhiều dòng trong CSS hoặc JS.

2️⃣ Cấu trúc cơ bản của một trang HTML

<!DOCTYPE html>  ← Khai báo đây là tài liệu HTML5
<html>           ← Thẻ bao toàn bộ trang web
  <head>         ← Phần đầu: chứa tiêu đề, mô tả, CSS, meta...
    <title>Tiêu đề trang</title>
  </head>

  <body>         ← Phần nội dung hiển thị trên trang
    <h1>Tiêu đề lớn</h1>
    <p>Đoạn văn bản.</p>
  </body>
</html>

3️⃣ Cách dùng CSS (trang trí giao diện)

CSS giúp đổi màu, phông chữ, bố cục...

body {
  background-color: lightblue; /* Màu nền */
  color: black; /* Màu chữ */
  font-family: Arial; /* Kiểu chữ */
}

4️⃣ JavaScript (tạo hành động cho web)

JavaScript giúp web tương tác như bấm nút đổi màu, hiển thị thông báo...

<button onclick="alert('Xin chào!')">Bấm thử</button>

5️⃣ Cách tải lên GitHub Pages

  1. Lưu file này tên index.html.
  2. Đặt vào thư mục gốc của repo GitHub.
  3. Push lên nhánh main.
  4. Vào Settings → Pages → Deploy from a branch → main → /(root).
  5. Chờ vài phút rồi truy cập đường dẫn web của bạn.
