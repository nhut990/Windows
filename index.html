Hướng Dẫn HTML và Ký Tự Cơ Bản

Trang này giúp bạn hiểu rõ từng ký tự trong HTML — ví dụ: <, >, &, "", {}, ()...

1️⃣ Giới thiệu các ký tự cơ bản trong HTML

  • < và > — Dấu ngoặc nhọn: dùng để mở và đóng thẻ HTML.
    Ví dụ: <p>Xin chào</p>
  • / — Dấu gạch chéo: dùng trong thẻ đóng. Ví dụ: </p>
  • "" hoặc '' — Dấu ngoặc kép/đơn: dùng để bao nội dung của thuộc tính. Ví dụ: <img src="hinh.jpg" alt='Ảnh minh họa'>
  • = — Dấu bằng: gán giá trị cho thuộc tính. Ví dụ: src="hinh.jpg"
  • & — Dấu & được viết thành &amp; trong HTML (để tránh bị nhầm là mã).
    Ví dụ: Viết &amp; để hiện ra &.
  •   — Khoảng trắng không ngắt dòng (non-breaking space).
  • { } — Dùng trong CSS hoặc JavaScript để chứa nội dung.
    Ví dụ: body { color: red; }
  • ( ) — Dấu ngoặc tròn, dùng trong hàm JavaScript.
    Ví dụ: alert('Xin chào')
  • ; — Kết thúc lệnh trong CSS hoặc JavaScript.
  • : — Dùng trong CSS để gán giá trị. Ví dụ: color: blue;
  • /* */ — Ghi chú trong CSS hoặc JS (trình duyệt bỏ qua).
  • // — Ghi chú 1 dòng trong JavaScript.

2️⃣ Cấu trúc cơ bản của một trang HTML

<!DOCTYPE html>  ← Khai báo loại tài liệu HTML5
<html>           ← Bắt đầu trang web
  <head>         ← Phần đầu: chứa tiêu đề, mô tả, CSS, meta...
    <title>Tiêu đề trang</title>
  </head>

  <body>         ← Phần nội dung hiển thị trên trình duyệt
    <h1>Tiêu đề chính</h1>
    <p>Đây là một đoạn văn bản.</p>
  </body>
</html>

3️⃣ Giải thích ký tự đặc biệt thường dùng trong HTML

  • &lt; → Hiển thị dấu <
  • &gt; → Hiển thị dấu >
  • &amp; → Hiển thị dấu &
  • &quot; → Hiển thị dấu "
  • &apos; → Hiển thị dấu '
  • &copy; → © Bản quyền
  • &nbsp; → Khoảng trắng (non-breaking space)

4️⃣ Kết hợp HTML + CSS + JavaScript

<!DOCTYPE html>
<html>
<head>
<style>
  body { background-color: lightyellow; }
  h1 { color: blue; }
</style>
</head>
<body>
  <h1>Xin chào!</h1>
  <button onclick="alert('Bạn đã bấm nút!')">Bấm vào đây</button>
</body>
</html>

5️⃣ Hướng dẫn tải lên GitHub Pages

  1. Lưu file này với tên index.html.
  2. Tải lên thư mục gốc của repository GitHub của bạn.
  3. Chọn nhánh main trong phần Settings → Pages.
  4. Chọn thư mục /(root) và bấm Save.
